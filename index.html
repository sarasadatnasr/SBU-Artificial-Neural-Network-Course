  • Introduction to Python

    The Introduction to Python Workshop offers participants a hands-on experience to learn the basics of Python programming language, covering essential topics such as syntax, data structures, control flow, and functions.

  • Introduction to Numpy

    The Introduction to NumPy Workshop is a comprehensive session that equips participants with the necessary skills to efficiently manipulate and analyze large arrays and datasets using the powerful numerical computing library, NumPy.

  • Introduction to Pytorch

    The Introduction to PyTorch Workshop offers participants a practical introduction to deep learning and PyTorch, enabling them to build and train neural networks for various applications.

  • Neural Network Architectures

    The Implementing Various Neural Network Architectures Workshop provides participants with hands-on experience in designing and implementing diverse neural network architectures, exploring different layers, activation functions, and connectivity patterns to tackle real-world machine learning challenges.
